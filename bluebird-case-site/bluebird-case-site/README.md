# Bluebird Case Public Website

This is a Create React App project prepared for easy deployment on Vercel.

## Run locally
```bash
npm install
npm start
```

## Build
```bash
npm run build
```

## Deploy on Vercel
1. Create a GitHub repo
2. Upload these files
3. In Vercel, click **Add New > Project**
4. Import the GitHub repo
5. Deploy

Vercel should auto-detect Create React App.
